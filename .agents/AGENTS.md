# VECTOR Engine - AI Agent Guidelines

## Goal of the Project
Develop a flexible, high-performance 3D First-Person Shooter (FPS) prototype featuring rigid body physics, robust ECS architecture, and a dynamic OpenGL rendering pipeline. The engine must remain decoupled from the game logic.

## Tools & Tech Stack
- **Language**: C++17 / C++20
- **Build System**: CMake (3.10+)
- **Graphics/Hardware API**: OpenGL 3.3 Core, SDL2, SDL2_ttf, SDL2_image
- **Physics**: Bullet3 (Bullet Physics)
- **Architecture**: Data-Oriented Entity-Component System (ECS) with variable-timestep loops
