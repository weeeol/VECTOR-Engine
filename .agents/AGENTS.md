# VECTOR Engine - AI Agent Guidelines

## Goal of the Project
Develop a flexible, high-performance 3D First-Person Shooter (FPS) prototype featuring rigid body physics, robust ECS architecture, and a dynamic rendering pipeline (OpenGL / DirectX). The engine must remain decoupled from the game logic.

## Tools & Tech Stack
- **Language**: C++17 / C++20
- **Build System**: CMake (3.10+)
- **Graphics/Hardware API**: OpenGL 3.3 Core, DirectX (11/12), SDL2, SDL2_ttf, SDL2_image
- **Physics**: Bullet3 (Bullet Physics)
